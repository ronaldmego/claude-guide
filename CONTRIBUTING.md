# Contributing

Thanks for your interest! This is a curated, **Spanish-language** enablement kit for
adopting Claude in teams.

## Ways to contribute

- Suggest **official / first-party** sources to add (Anthropic/Claude docs, guides, videos).
- Improve the templates (`CLAUDE.md` / `settings.json`) or the session plan.
- Fix errors, broken links, or outdated metrics.

## Ground rules

- **Content is in Spanish** (by design — that's the niche). Keep content contributions
  in Spanish; this `README` / `CONTRIBUTING` stay in English for discoverability.
- Prefer **official / first-party** sources; mark third-party clearly.
- **Keep it generic and reusable** — no secrets, no personal data, no
  employer-specific content. Templates must stay sanitized (placeholders, never real values).

## How

1. Open an issue describing the change (bug report or feature/source-suggestion templates).
2. Fork, create a branch, make the change.
3. Open a PR referencing the issue. Keep PRs focused on one thing.
